To bet:

Note: Check !state first. See commands for details. WhiskChat provides a nice command to bet (/bet) as well as a nice interface. Try it! Syntax: 

/tip WhiskDiceBot (amount) [percentage]

Note that percentages are limited from 1% to 75% to prevent draining, and the max bet is 1 mBTC. Examples:

 /tip WhiskDiceBot 0.25

 /tip WhiskDiceBot 0.25 60%

WhiskDiceBot also supports nice messages:

 /tip WhiskDiceBot 0.25 Hm, I'll take a 75% please.

However percentages below 10% must be written like this:

 /tip WhiskDiceBot 0.25 01%

Commands

Before you bet, you should always check if the bot is up and running to avoid disappointment. Use the !state command.

Want to check if you've made a profit? Use !history.

Extra Nice Features

  • If you are the last winner of a bet, and someone else's (or your!) bet loses, there is a chance of you winning half his/her bet as a bonus!
  • There are other bots in #botgames! Check out !bots for info.

Understanding Win/Loss Messages

✔ whiskers75 won 0.22 mBTC! (-3.82 mBTC) [50%, 1.94x: 44 < 51]

This means:

✔/✗ whiskers75 won/lost (amount) mBTC! ([total profit/loss mBTC) [(chance)%, (payout)x: (rolled) < (the number rolled needs to be under to win)]
